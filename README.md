## ✨ Key Features

- 🌍 **Full i18n Support**: Available in English, Castellano (Spanish), and Català (Catalan).
- 🔗 **Deep Linking**: Direct access to specific challenges via URL hashes (`/#slug`).
- 💻 **Exploit Payloads**: Integrated Python solvers, XPath payloads, and technical scripts with copy-to-clipboard functionality.
- 📱 **Responsive Design**: Glassmorphic, neon-themed UI optimized for all devices.
- 🎯 **Multi-Flag Support**: Dynamic rendering for challenges with multiple tasks (OSINT, forensics).

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [pnpm](https://pnpm.io/) (recommended) or npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Fedeerich/writeup-soterctf.git
   cd writeup-soterctf
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm run dev
   ```

## 🏗️ Technical Architecture

- **Core**: React 18 + Vite
- **Styling**: Vanilla CSS with custom design tokens for a "hacker" aesthetic.
- **Icons**: Lucide React
- **State Management**: React Context API for localization and global application state.
